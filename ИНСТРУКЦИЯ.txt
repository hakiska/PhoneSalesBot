============================================
  Бот учёта продаж — Инструкция
============================================

1. УСТАНОВКА (один раз)
   - Скачай Python с https://www.python.org/downloads/
   - При установке ОБЯЗАТЕЛЬНО поставь галочку Add Python to PATH !!!
   - Перезагрузи ноутбук

2. ЗАПУСК
   - Два раза кликни на start_bot.bat
   - Откроется чёрное окно — НЕ ЗАКРЫВАЙ его пока работаешь
   - Бот работает пока окно открыто

3. КАК ПОЛЬЗОВАТЬСЯ (в Telegram)
   Пиши боту продажи как в тетрадь:

   Note 9s 1 * 9500 нал
   11 Pro GX original 1 * 11000 К
   A54 2 * 8000 Д

   Оплата:
     нал — наличные
     К   — Каспи Камиль
     Д   — Каспи Диана
     Р   — Каспи Рауф
     Ра  — Каспи Разия
     ИП  — Каспи ИП

4. КОМАНДЫ В БОТЕ
   /report  — посмотреть продажи за сегодня
   /excel   — скачать Excel-файл за сегодня
   /excel 2026-04-08 — Excel за конкретную дату
   /ret     — возврат товара (выбрать из списка)
   /cancel  — удалить последнюю запись
   /help    — справка

5. ЗАВЕРШЕНИЕ РАБОТЫ
   - Закрой чёрное окно или нажми Ctrl+C
   - Все данные сохранены, ничего не потеряется
